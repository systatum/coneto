# Coneto: A React UI Library for Every Use Case, Done Right

Deliver apps in half the time with Coneto, free of charge! Coneto is _the_ extremely complete React UI library created by <a href="https://systatum.com">Systatum</a>. It is designed with modernity, simplicity, and usability at its core.

Here you’ll find everything from stateful forms you can build in just a few lines of code to a versatile data grid and a solid set of tabs and view management tools. A complete example and self-documented code show how to use those components across different scenarios, so you can build polished apps even faster.

Our components are organized into four primary categories:

- **Controls**: Interactive components that typically trigger page re-renders.
- **Content**: Components for displaying or structuring data/information in a clear, accessible way.
- **Input**: Components that allow users to submit data or express intent. These support the `onChange` callback.
- **Stage**: Layout and container components that provide structure and presentation for the other elements.

Coneto is battle-tested, having been used across many projects in both React and Electron, and it consistently proves to be both reliable and visually polished in each environment. So, you’re definitely welcome to use it, and please give us, <a href="https://systatum.com">Systatum</a>, proper credit when you can. Also, feel free to voice out your ideas or concerns, if any.

## Logo

The word *logo* traces back to the Ancient Greek *logos* - "word," "speech,"
"reason," a root that points to meaning as much as expression. With that in
mind, the name Coneto reflects a simple idea: *components to connect*.
Coneto presents a collection of enterprise-grade UI components
designed to close the gap between what you envision and what you ship, where
each piece serves that connection quietly and precisely, without friction, so
attention remains on your product rather than its implementation. Even this idea is
carried through the visual mark itself, where repeating arcs of the letter "C"
interlock and draw inward, guiding the eye toward a central square. That
center represents the user - the origin and destination of every interface -
a reminder that everything ultimately connects back to you and the people your
product serves.
