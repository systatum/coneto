# Installation Guide

This guide explains how to install and use Coneto in your application.

Install the package:

```bash
npm install @systatum/coneto
```

or

```bash
pnpm add @systatum/coneto
```

or

```bash
bun add @systatum/coneto
```

## Usage

Import components directly from their module path:

```tsx
import { Button } from "@systatum/coneto/button";

function App() {
  return <Button>Save</Button>;
}
```
